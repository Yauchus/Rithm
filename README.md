# 2D Rhythm Game

# Мини-прототип 2D ритм-игры на Unity, где игрок должен нажимать клавиши в такт музыке и попадать в ноты.

# Описание проекта
# Игровой процесс строится вокруг падающих нот и оценки точности нажатия. Ноты движутся по сцене синхронно с музыкой, а результат попадания отображается на экране.

# Правила игры
# Нота считается успешно взятой, если нажать нужную клавишу в момент попадания.
# За попадание выдаётся одна из оценок: Perfect, Good, Normal или Miss.
# Если игрок нажимает не вовремя или пропускает ноту, засчитывается Miss.

## Управление
# `A` — левая дорожка.
# `S` — вторая дорожка.
# `D` — третья дорожка.
# `J` — четвертая дорожка.
# `K` — пятая дорожка.
# `L` — правая дорожка.
